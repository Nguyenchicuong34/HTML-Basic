##  Overview
Welcome to the source code of **[HTML BASIC]**.
This project is a static website designed to demonstrate structure, clean coding practices, and responsive design using core web technologies.

> **Live Demo:** https://github.com/Nguyenchicuong34/HTML-Basic

##  Key Features
* **Semantic HTML5:** Built with SEO-friendly structure and accessibility in mind.
* **Responsive Layout:** Fully compatible with Desktops, Tablets, and Mobile phones.
* **Clean Code:** Organized and commented code for easy understanding.
* **Performance:** Lightweight and fast loading without heavy frameworks.
